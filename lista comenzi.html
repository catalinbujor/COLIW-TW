Lista comenzi

web twitter follow "utilizator"

  • Aceasta comanda ofera accesul la serviciul de follow a unui utilizator specificat pentru platforma twitter.

web join rss -where -unique | web wordpress publish -as HTML

  • Prin intermediul acestei comenzi utilizatorul obtine o lista de fluxuri de stiri in vederea publicarii in format HTML pe un blog (ex:Wordpress).

web flickr photos -tag:Tux -first | web instagram update user

  • Prin intermediul acestei comenzi utilizatorul isi actualizeaza fotografia de profil de pe Instagram pe baza unei fotografii preluate de pe Flickr.

web gmail -tag:Important -since:2018-01-01 > msg.RSS

  • Cu ajutorul acestei comenzi utilizatorul isi poate accesa email-ul si poate solicita diverse informatii cu privire la acesta.
